# Кошкин Дом — сайт отеля (прототип)

Статический сайт-визитка отеля «Кошкин Дом» (Зеленоградск). Свёрстан по дизайну из Figma,
контент повторяет исходный сайт koshkin-dom.ru. Прототип для показа заказчику —
без интеграции с бэкендом/API.

## Как запустить локально

Просто откройте `index.html` в браузере. Либо поднимите любой статический сервер:

```bash
# Python
python3 -m http.server 8000
# или Node
npx serve .
```

и откройте http://localhost:8000

## Как залить на сервер

Это обычный статический сайт — загрузите всю папку целиком на любой хостинг
(Nginx/Apache, Netlify, Vercel, GitHub Pages, Beget, timeweb и т.п.).
Точка входа — `index.html`. Ничего собирать не нужно.

## Структура

```
index.html            — единственная страница (все секции)
css/
  fonts.css           — подключение шрифтов (@font-face)
  styles.css          — стили и дизайн-система
js/
  main.js             — меню, галерея-лайтбокс, форма брони, анимации
assets/
  logo/               — логотип (SVG)
  fonts/              — Cormorant Garamond + Jost (self-hosted, OFL)
  img/                — фотографии номеров, галерея, достопримечательности, карта
```

## Что реализовано

- Адаптивная вёрстка (десктоп / планшет / мобильный), бургер-меню
- Плавная навигация по якорям, «липкая» шапка
- Галерея с лайтбоксом (стрелки, Esc, клик по фону)
- Модальное окно брони (демо-форма без отправки)
- Строка бронирования с выбором дат и гостей
- Анимации появления секций при скролле (учитывают prefers-reduced-motion)

## Важно (прототип)

- Формы брони и «Подробнее» открывают демо-окно и **не отправляют данные** —
  это заглушки под будущую интеграцию с API.
- Онлайн-камера и карта ведут на внешние сервисы (kgd.ru, Яндекс Карты).
- Цены, тексты и отзывы соответствуют исходному сайту на момент вёрстки.
- Шрифты подключены локально (в `assets/fonts/`), поэтому сайт не зависит от
  Google Fonts и корректно отображается в т.ч. при ограничениях доступа к CDN.

## Шрифты

Cormorant Garamond и Jost — свободные шрифты под лицензией SIL Open Font License 1.1
(тексты лицензий — в `assets/fonts/`).
